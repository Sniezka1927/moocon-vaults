## Points

Points are the protocol's engagement layer. They track your participation over time and are a bonus reward for all active stakers — separate from drawing tickets.

### Stake Points

Earned by holding pTokens. The longer and larger your deposit, the more stake points you accumulate. Points are processed continuously.

### Referral Points

Earned when users you referred are active in the protocol. When someone uses your referral code and deposits, their activity generates referral points that flow back to you.

### Multipliers

Each user has a points multiplier (default 1×). Multipliers increase both the amount of points you earn **and** the number of tickets you hold — giving you better drawing odds alongside higher point accumulation.

## Referral Program

### Creating Your Referral Code

Once you have a qualifying deposit, you can generate a unique referral code from the dashboard.

- Click **Create Referral Code** in the referral panel.
- Sign the message with your wallet (Ledger supported).
- Your code is registered on-chain and ready to share.

You can share your code as a link or via the referral card in the app.

### Using a Referral Code

When signing up, new users can enter a referral code before or after depositing.

- Navigate to the referral panel.
- Enter the code and sign the verification message with your wallet.
- The link between your wallet and the referrer is recorded.

**Rules:**
- Each wallet can only use one referral code (one-time action).
- You cannot use your own referral code.
- Using a code earns you no immediate bonus, but the referrer earns referral points from your activity.
